Réseau Social pour l'atelier final Ifocop (Rouzaud Pierre)

Pour le fonctionnement du site ajouter le fichier config.js à la racine du dossier back
avec pour variables:
* devEnv 
* clientUrlDev 
* clientUrlProd
* secret 
* tokenExpire 
* secretResetPasswordToken 
* resetPasswordTokenExpire 
* email
* emailPassword
